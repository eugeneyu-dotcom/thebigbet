自從埃及跟阿根廷的比賽之後，各種關於裁判不公的說法就甚囂塵上。不僅埃及的球員與教練在賽後訪談提及，埃及足協也正式提出了抗議。大批「陰謀論」的網友更是湧入 FIFA，或在網路上大肆傳播各種裁判偏袒阿根廷的說法，並附上幾張截圖，繪聲繪影地描述埃及是如何受到不公平的對待。但為避免版權爭議，這邊就不放圖片了，有興趣的網路上很容易搜索到。

## 憑心而論，這場哨音確實比較偏向阿根廷

但其實，看完整場，憑心而論，裁判確實是比較偏向阿根廷沒錯。一些 50/50 的吹判都對阿根廷比較有利，阿根廷獲得的第一顆點球（雖然被擋下了）我覺得是否有必要吹罰到點球也有爭論的空間。而裁判相對默許阿根廷使用南美足球那更為粗暴的身體接觸標準，也確實是讓阿根廷更為有利。

## 但這樣就能說裁判操控了比賽嗎？我持反對意見

可光憑這幾點，可以說裁判操控了比賽嗎？我其實持反對意見。

阿根廷確實身體接觸上比較粗暴，但南美足球一向如此，而在裁判沒有吹罰的狀況下使用更多的身體接觸，本來就是很正常的一件事，這點應該是埃及球員自己必須去適應尺度。就像法國對巴拉圭那場，[Mbappé](wiki:Kylian_Mbappé) 明確在賽後說出，如果他們想要踢髒足球，那我們也奉陪——這樣的態度才是面對世界盃正確的態度。

而在其他爭論中，最多人討論的兩個點，一個是埃及原本的第二顆進球因為 VAR 回看在搶球時有犯規而取消，另一個是 [Julián Álvarez](wiki:Julián_Álvarez) 在禁區內絆倒 [Salah](wiki:Mohamed_Salah) 那球到底有沒有犯規。

## 先澄清一個常見誤區：VAR 沒叫主裁看，不代表 VAR 沒介入

而這，就要說到 VAR 相關的介入規則了。首先澄清一個常見的誤區——並不是 VAR 沒有回放、或沒叫主裁過去看，就代表 VAR 沒有介入。

事實上，如果 VAR 回看之後和主裁判意見一致，覺得主裁的判罰沒有問題，VAR 是不會通知主裁的，但 VAR 還是會確實在每一個指定的環節回看。本屆中，VAR 明確可以介入的時間點如下：

### 1. 進球與否（Goals / No Goals）

這是 VAR 最頻繁介入的時機。當進球發生時，VAR 會自動在後台檢查進球的整個「進攻發起階段」（Build-up phase）是否有任何違規。

- **檢查重點：**
  - 進球前是否有越位（Offside）。
  - 進攻方在進球前是否有犯規（如拉扯、推人、手球）。
  - 在進攻發起前，球是否已經先出界。
  - 球是否已經完全越過球門線（若該賽事無門線技術）。

### 2. 點球判罰（Penalty / No Penalty）

禁區內的犯規往往直接決定比賽勝負，因此任何點球的判罰或漏判，VAR 都會進行介入審查。

- **檢查重點：**
  - **犯規地點：** 犯規究竟發生在禁區內還是禁區外（這決定了是點球還是自由球）。
  - **是否真有犯規：** 防守方是否有實質性的犯規動作，還是進攻方「假摔」（Diving / Simulation）。
  - **手球爭議：** 防守方在禁區內的手球是否構成點球條件（如手臂是否處於不自然擴大的位置）。
  - 主裁判漏判了明顯的禁區內犯規。

### 3. 直接紅牌（Direct Red Cards）

VAR 會審視場上可能構成「直接驅逐出場」的嚴重犯規或暴力行為，確保沒有球員逃過懲罰，也確保沒有球員被冤枉罰下。

- **檢查重點：**
  - **嚴重犯規（Serious Foul Play）：** 例如亮鞋釘的惡意飛鏟、過度使用武力。
  - **暴力行為（Violent Conduct）：** 球員在死球狀態下或無球狀態下的打架、報復性動作、咬人或吐口水。
  - **破壞明顯進球機會（DOGSO）：** 防守方是否在最後一道防線上犯規，破壞了對手必進的單刀球。
- **⚠️ 絕對盲區（重要例外）：** VAR 不能介入審查「第二張黃牌」。如果一名球員因為一張充滿爭議的黃牌而累積兩黃變一紅被罰下，VAR 無權介入推翻那張黃牌。

### 4. 認錯球員（Mistaken Identity）

這種情況在現代足球非常罕見，但依然被列入規則中。

- **檢查重點：**
  - 當球場上發生混亂時，主裁判如果向錯誤的球員出示了黃牌或紅牌（例如雙方長得很像，或是背號看錯），VAR 會介入提醒裁判糾正對象，避免無辜球員受罰。

## 埃及被吹掉的第二球：完全符合第一點，是正確的吹判

其中，埃及被吹掉的第二球，就完全符合第一點中的第二項：**進攻方在進球前是否有犯規（如拉扯、推人、手球）。**

這球通過回放，我們可以很明顯地看到埃及確實是依靠一個踩腳的犯規動作才獲得球權，這也導致他們後續一系列的進攻應該都是無效的，當然，包括最後進球的結果。因此這其實是一個正確的吹判。

有人會說是不是隔太久了？但這就牽扯到 VAR 介入時機了。VAR 只有在「進球」之後，才會去回看整個進攻流程中是否有犯規，而這球中間沒有任何的球權轉移，是一次完整的進攻流程，且最後進球了，因此才會觸發回看流程。所以你會覺得很久，是因為必須等到確認「進球」後 VAR 才會出來確認整個流程；若是沒有進球，這甚至不是一次 VAR 會去回看的動作。

## Julián Álvarez 絆倒 Salah：VAR 一定看了，判斷與主裁一致

再來，很多人提到的 Julián Álvarez 在禁區內絆倒 Salah 那球，這球在禁區邊緣，因此明確符合第二點中的 1、2、4 項。因此可以確定的是，VAR 肯定有回看這一球，並確認是否要給點球。

但最後 VAR 並沒有通知主裁去看回放，因此基本上可以確定，這球 VAR 的判斷與主裁判一致，並不構成一個犯規。而若我們看慢放影片，也可以很明顯看到，Julián Álvarez 確實是先碰到了球，接著收腳避免接觸，後續的接觸是由於 Salah 自己繼續向前之後造成的，且這個接觸非常輕微，沒有如同埃及第二球時那種踩踏的動作，因此不構成犯規是完全可以理解的。

## 結語：疑點確實多，但這兩球真的沒問題

當然，這屆的 FIFA 規則有很多值得爭論的地方，例如 [Balogun](wiki:Folarin_Balogun) 的紅牌延期，所以懷疑陰謀論確實可以理解。畢竟這屆說不清楚且疑似的地方太多，就連這場，雖然爭議最多的兩個判決沒有問題，但你要說整場哨音比較偏向阿根廷，那就連身為 [Messi](wiki:Lionel_Messi) 粉的我也只能說，確實，是有這麼回事，你說得對。

至於其他的爭論，諸如阿根廷歷來犯規次數多但黃牌少，這個我個人覺得就更好笑了。犯規次數多少跟黃牌量本來就沒有絕對關聯，你可以說阿根廷更喜歡犯一些單純噁心人、無意義的小規，但拿黃牌量說事，這個就有點扯遠了。
