這屆世界盃因為擴軍到48隊、12個小組，晉級規則比過去複雜不少——不再是單純「小組前兩名晉級」這麼簡單，賽程表上有「A/B/C/D/F 3」這種過去沒有的標示。從網上找到一張畫得很清楚的賽程圖，整理一下晉級相關規則，方便大家看懂自己支持的隊伍接下來可能踢誰。

## 晉級門檻：前兩名穩了，剩下8張門票要搶

12個小組，每組前兩名自動晉級32強淘汰賽，這部分沒有疑問，總共24個名額。剩下的8個名額，則是從12支小組第三名球隊裡，挑出成績最好的8支，一起加入晉級行列。

換句話說，本屆就算只拿到小組第三也不必絕望，還沒有出局——還要看其他11個小組的第三名打得怎麼樣，跟他們比成績。我個人十分喜歡這個設計，這讓第三名球隊也有更多爭勝的動力和進攻的動力，不會輕易放棄比賽。

## 排名怎麼算：積分、淨勝球、進球數

12支小組第三名要怎麼排出誰是「最佳8支」，依序看三個指標：

1. **積分（PTS）**
2. **淨勝球（GD）**
3. **進球數（GS）**

積分一樣的話比淨勝球，淨勝球也一樣才比進球數。跟小組內排名的邏輯相同，只是這次是拿來跟其他組的第三名互相比較。

## 觀賽重點：賽程表早就埋好了「誰可能踢誰」

這是最有趣，也最容易讓人霧裡看花的部分。

由於要等到所有小組賽踢完才會知道哪8支隊伍能以第三名晉級，賽程表沒辦法事先寫死「A組第三 vs B組第三」這種固定對戰——因為A組第三可能根本沒晉級。FIFA的做法是：先把32強的16場對戰位置排好，其中8個位置直接是兩支確定球隊的對戰（例如小組第一對小組第二），另外8個位置則保留給「最佳第三名」，並且預先限定這個位置只能由特定幾個小組的第三名來填補。

實際整理出來的對戰範圍如下：

| 小組第一 / 第二 | 對手限定為以下小組的第三名 |
|---|---|
| E組第一 | A、B、C、D、F 組第三 |
| I組第一 | C、D、F、G、H 組第三 |
| D組第一 | B、E、F、I、J 組第三 |
| G組第一 | A、E、H、I、J 組第三 |
| A組第一 | C、E、F、H、I 組第三 |
| L組第一 | E、H、I、J、K 組第三 |
| B組第一 | E、F、G、I、J 組第三 |
| K組第一 | D、E、I、J、L 組第三 |

### 為什麼每個位置只能由「特定幾組」的第三名來填？

這就是跟分區規則有關的地方了。

12組第三名中最終會有8組晉級，但到底是「哪8組」，在小組賽全部踢完前是未知數，而且光是排列組合就有數百種可能。FIFA的解法，是賽前就準備好一張完整的分配表（也就是分區規則）：針對每一種「哪8組第三名晉級」的組合，都預先規定好這8支球隊分別該填進哪一個對戰位置。

這張表的設計有兩個核心原則：

- **避免同組重逢**：盡量不讓在小組賽已經交手過的球隊，在32強又立刻碰頭。
- **維持籤運平衡**：讓晉級的第三名盡量平均分散到籤表的各個區塊，而不是全部擠在同一邊。

也正因為要同時滿足這兩個條件，每個保留給第三名的位置，才會被事先框定只能由某幾個特定小組的第三名來填補。賽程表上「A/B/C/D/F 3」這樣的標示，就是這套分配表替該位置劃定的候選範圍——最終由其中哪一組遞補，要等8席第三名全部確定後，再對照分配表才能定案。

剩下的8場32強對戰，則是雙方身分都已確定的標準對戰，例如小組第一對小組第二、或是兩個小組的第二名互相對戰，不受第三名名單的影響。

## 對最後一輪比賽的影響？

理解這套規則之後，看小組賽最後一輪會更有意思——尤其是那些已經提前確定晉級、但還在小組第一和第二之間擺盪的球隊。畢竟，搶到小組第一未必是最好的選擇：如果小組第二之後要面對的對手，反而比小組第一要面對的「最佳第三名候選」更弱，那麼故意保守求穩、退而求其次拿第二名出線，也不是不可能出現的策略。這也是為什麼最後一輪小組賽，賠率往往會比想像中更難看懂的原因。
