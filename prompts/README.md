# The Big Bet — 賽事分析文章撰寫系統

本資料夾包含 The Big Bet 網站的**文章潤稿與 SEO 工具包**。  
適用對象：所有參與撰寫賽事分析文章的協作者。

---

## 這個系統在做什麼

你負責寫分析內容，AI 負責：
- 潤稿、校正語法、改善流暢度
- 補齊文章結構（地主優勢、賠率解讀等段落）
- SEO 標題與 meta description 優化
- 輸出可直接存檔上線的 MDX 格式

**重點：AI 會學習你的寫作風格。** 每次潤稿後，系統會建議新的風格觀察，你確認後存入自己的風格檔，下次就更像你本人寫的。

---

## 資料夾結構

```
prompts/
├── README.md                      ← 你正在看的這份說明
├── analysis-article-writer.md     ← 主要 Prompt（複製貼入任何 AI 工具）
└── author-styles/
    ├── _template.md               ← 新作者的空白風格範本
    └── eugene.md                  ← Eugene 的個人風格檔（範例）
```

---

## 第一次使用：新作者設定

### 步驟 1：建立你的個人風格檔

1. 打開 `author-styles/_template.md`
2. 複製一份，改名為你的名字，例如 `author-styles/john.md`
3. **不需要現在填寫任何內容**，讓它保持空白，AI 會從你的第一篇草稿自動提取

> 風格檔的內容會隨著你投入的草稿越來越多而逐漸豐富，通常 3–5 篇後就能達到很好的效果。

## 日常使用流程

### 方式 A：使用 Claude Code（推薦，最省力）

1. 在 Claude Code 對話中，直接說：「幫我潤稿」或貼上草稿
2. Claude 會自動觸發 `the-big-bet-analysis` skill
3. 告訴 Claude 你的作者名稱，它會直接讀取你的風格檔
4. 潤稿完成後，Claude 會建議更新風格檔——確認後它會自動寫入，不需要手動操作

### 方式 B：使用 ChatGPT、Gemini 或其他 AI 工具

**每次使用前，依序貼入以下三樣東西：**

**① 複製主要 Prompt**  
打開 `analysis-article-writer.md`，複製「System Prompt」區塊內的全部內容，貼到 AI 的 System Prompt 欄位（或對話最前面）。

**② 貼上你的風格檔**  
打開 `author-styles/<你的名字>.md`，複製全部內容，在對話中說：
> 「這是我的作者風格檔：[貼上內容]」

如果風格檔是空的或第一次使用，直接跳過這步也可以。

**③ 貼上草稿，開始處理**  
直接貼上你的分析草稿，AI 就會開始工作。

---

## 潤稿結束後：更新風格檔

每次處理完文章，AI 會提出「風格觀察建議」，例如：

```
【風格觀察建議】
- 新發現：你習慣用「但整體來說...」作為段落轉折
- 建議加入典型句子：「直到傷停補時才靠著一個有點幸運成分的頭槌破門」
```

**Claude Code 用戶**：直接確認，AI 會自動更新 `author-styles/<你的名字>.md`。  
**其他 AI 工具用戶**：手動將建議的內容貼入 `author-styles/<你的名字>.md` 對應的區塊中儲存。

## 文章存檔位置

潤稿完成後，將 MDX 內容存入：
```
src/content/analysis/zh-tw/<檔名>.mdx
```

檔名規則：`主隊英文-vs-客隊英文.mdx`，例如 `canada-vs-qatar.mdx`

存檔後執行 `npm run dev` 可在本地預覽，圖表（H2H 柱狀圖、五維雷達圖）會自動從文章的 frontmatter 資料渲染，不需要在文章裡寫任何程式碼。

## 常見問題

**Q：草稿要寫多完整？**  
A：沒有硬性要求。只有幾個要點也可以，AI 會補齊結構。但你提供的分析觀點越完整，產出的文章就越有你的個人特色，也越準確。

**Q：比分預測和信心指數需要填嗎？**  
A：不需要。AI 會從你的文章語氣自行推斷，並在輸出後說明它的推斷依據，你可以確認或要求修改。

**Q：賠率需要自己查嗎？**  
A：不需要。如果你在 Claude Code 環境中操作，可以要求 Claude 幫你查詢當前賠率。如果自己使用其他 AI 工具，可以先查好再提供，或填「待確認」。

**Q：可以寫英文文章嗎？**  
A：目前系統設定以繁體中文輸出。如需英文版本，在貼給 AI 時另行說明即可。

**Q：第一次使用，風格檔是空的，這樣可以嗎？**  
A：完全可以。AI 會從這次草稿即時提取你的寫作風格，一樣能產出不錯的結果。處理完後記得把 AI 建議的風格觀察存入你的風格檔，下次就會更準確。
